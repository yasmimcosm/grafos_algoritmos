# 📊 Trabalho Final – Teoria dos Grafos e Algoritmos

Este repositório contém a implementação do **Trabalho Final da disciplina de Grafos / Algoritmos**, envolvendo diferentes problemas clássicos da Teoria dos Grafos e análise de desempenho computacional.

## 📌 Descrição do Projeto

O trabalho consiste na implementação e análise experimental de algoritmos aplicados a problemas clássicos em grafos, com foco em desempenho computacional e qualidade de solução.

- 📮 **Problema do Carteiro Chinês**
  - Grafos não-eulerianos
  - Resolução do 1-matching (heurística ou enumeração completa)
  - Avaliação de esforço computacional

---

- **Como Rodar? **
  - Estando na pasta /src, digite o seguinte comando:
  ```bash
  g++ main.cpp Grafo.cpp GeradorGrafo.cpp PccSolver.cpp -o programa
  ``` 

  Todos os arquivos sem ser a main podem ser compilados ou não. Depende das funcionalidades que a main usa.
